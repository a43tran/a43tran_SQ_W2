# Week 2 Example 2: Platformer with Platforms Array

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.
To play via GitHub Pages, visit the link provided on the submission page.

Move the player in your desired direction by pressing the arrow keys or WASD.
Jump by pressing the UP arrow or W. 

**Controls:**
Move the player left or right by pressing the LEFT or RIGHT arrow keys or A/D keys.
Jump by pressing the UP arrow or W keys.
